## About <a name = "about"></a>

There are 2 config files in use. 
1. base_config.py which helps setup all paths to be used.
2. pipeline_config.py which helps setup the experimental parameters for the pipeline.

## Usage <a name = "usage"></a>

### base_config.py
Specific usernames can be established by adding usernames to the dictionary.
Default folder paths can be modified to suit the user's folder structure.
